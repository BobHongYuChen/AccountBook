# AccountBook
项目描述：一款记账APP，能够分类展示用户每天的账目，可以对账目进行相关的增删查改

项目使用的控件和布局：

​					TextView

​					Floating Button

​					ViewPager

​					Fragment

​					ListView

​					Dialog

​					EditText

​					RecyclerView

​					Table Layout

​					AlerttDialog

设计模式：单例模式、适配器模式

数据库：SQLite
